# Pronostix

MVP modular para quiniela del Mundial FIFA 2026 usando GitHub Pages + Supabase.

## Configuración

Edita `config.js` y reemplaza:

```js
const SUPABASE_URL = "TU_PROJECT_URL";
const SUPABASE_ANON_KEY = "TU_ANON_KEY";
```
